# Pipeline APIs

## Purpose
Expose APIs to submit and manage runs.
Expose APIs to fetch status, logs, evidence, and lineage.

## Run APIs
- Submit run
- Cancel run
- Get run status
- Get run logs
- Get run evidence

## Metadata APIs
- List contracts
- List runs
- Query lineage

## Webhooks
- Run started
- Run succeeded
- Run failed
- SLO breach
